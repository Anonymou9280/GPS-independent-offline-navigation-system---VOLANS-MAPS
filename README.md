## 📱 What VOLANS MAPS Does

VOLANS MAPS is an offline navigation app that estimates your location without relying on GPS or internet access. It uses built-in device sensors such as the accelerometer, gyroscope, and magnetometer to track movement and direction. It then uses Kalman filtering to smooth the sensor data and improve position estimates.

This makes the app useful in places where GPS is weak or unavailable, such as:
- Underground areas
- Dense city streets
- Remote roads
- Indoor spaces
- Areas with no mobile data

## 🗺️ How It Works

VOLANS MAPS combines sensor readings to estimate where you are. Each sensor gives part of the picture:
- The accelerometer tracks movement
- The gyroscope tracks rotation
- The magnetometer helps with heading direction

The app blends this data with Kalman filtering. That process helps reduce noise and gives a more stable position estimate. It does not need a live GPS signal to work, which is useful when other navigation apps fail.
